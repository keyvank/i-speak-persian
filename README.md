# I speak Persian!

قشنگ‌تر نیست که توی صحبت‌های فنی‌مون هم از لغات و اصطلاحات فارسی استفاده کنیم؟ این مخزن تلاشی برای اینکه ما رو با برابرهای خوب فارسی برخی از واژگان انگلیسی روزمره که استفاده می‌کنیم آشنا کنه.

**_اگه با ترجمه یا جایگزین یک لغت مشکل دارین، یک Issue (جستار) گیتهابی بسازین و لینکش رو جلوی اون لغت قرار بدین 🙏_**

سوالاتی که پیش میاد:

1. کلماتی مثل Product Manager یا User Experience با اینکه معادل فارسی خوبی دارن، ولی نسخه کوتاه‌شده جالبی ندارن. تکلیف چیه؟
2. می‌توانیم از [واژه‌نامهٔ انجمن اوبونتو](https://wiki.ubuntu-ir.org/wiki/%D9%88%D8%A7%DA%98%D9%87%E2%80%8C%D9%86%D8%A7%D9%85%D9%87%D9%94_%D8%A7%D9%86%D8%AC%D9%85%D9%86) برای تکمیل این مخزن استفاده کنیم.
